HTML and CSS
I started by structuring my resume in HTML, using semantic tags for clarity and accessibility. Then, I applied CSS to enhance the visual appeal of the webpage, transforming the raw HTML into a styled, user-friendly site.
Deploying as a Static Website
Once I was happy with my HTML and CSS, I deployed my resume online as an Azure Storage static website. While services like Netlify and GitHub Pages are great, I chose Azure Storage for this project to get hands-on experience with it.
Visitor Counter with Javascript and CosmosDB
I wanted to track how many people visited my site, so I added a visitor counter using Javascript. The counter retrieves and updates its count in a database. For this, I used the Table API of Azure’s CosmosDB. It was my first time working with CosmosDB, and I found it to be a powerful and flexible database service.
API with Azure Functions
To ensure secure communication with CosmosDB, I created an API using Azure Functions with an HTTP trigger. This API accepts requests from my web app and communicates with the database. It was a great way to learn about serverless architecture and how it can simplify backend development.
Infrastructure as Code
Instead of manually configuring my API resources, I defined them in an Azure Resource Manager (ARM) template. This approach, known as “infrastructure as code” (IaC), saves a lot of time and reduces the risk of human error.
Source Control and CI/CD
I used GitHub for source control and set up continuous integration and deployment (CI/CD) for both my back-end API and front-end website. With this setup, my application updates automatically whenever I make a change to the code. It’s a huge time-saver and ensures my website is always up-to-date with the latest changes.
Conclusion
Building this project was a fantastic learning experience. I got hands-on experience with many different technologies and best practices, and I’m excited to apply what I’ve learned to future projects. I hope my journey can inspire and help others who are looking to do something similar.
